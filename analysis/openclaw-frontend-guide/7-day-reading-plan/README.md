# OpenClaw 7 Day Reading Plan

这是一套按天拆分的源码阅读计划，目标是帮助你快速建立：

- 当前项目架构脑图
- 完整主流程认知
- 可落地的扩展与定制思路

建议顺序：

1. `day-01-cli-and-entry.md`
2. `day-02-onboard-and-config.md`
3. `day-03-gateway-and-control-plane.md`
4. `day-04-routing-and-sessions.md`
5. `day-05-agent-execution.md`
6. `day-06-channels-and-messaging.md`
7. `day-07-plugins-ui-and-extension.md`

每天建议固定输出：

- 一张简单模块图
- 一份 5-10 条阅读笔记
- 对当天“必须回答的问题”的回答
- 一页你自己的扩展思考
